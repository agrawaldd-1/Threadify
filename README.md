# 🛍️ Threadify – Modern Fashion E-Commerce Frontend

Threadify is a modern fashion e-commerce website built using HTML, CSS, and JavaScript. The project focuses on creating a clean, responsive, and interactive shopping experience inspired by modern fashion platforms.

## ✨ Features

### 🔍 Smart Search Suggestions

* Real-time search suggestions
* Category-based product search
* Instant filtering using JavaScript

### 👤 Account Menu

* Interactive account dropdown
* Profile section
* Orders section
* Settings panel
* Logout option

### 📂 Sidebar Navigation

* Slide-in category menu
* Fashion categories including:

  * Mens
  * Womens
  * Kids
  * Gen Z
  * Footwear
  * Watches
  * Bags
  * Jewellery
  * Sports Wear
  * Party Wear
  * Street Wear
  * Sale
  * New Arrivals

### 🎨 Hero Banner

* Modern gradient design
* Fashion-focused branding
* Call-to-action buttons
* Promotional section

### 🎯 Offer Slider

* Interactive image slider
* Multiple promotional offers
* Previous / Next navigation controls

### 👔 Men's Fashion Collection

* Product cards
* Pricing section
* Available color variants

### 👗 Women's Fashion Collection

* Fashion product showcase
* Color options
* Responsive product grid

### ⚡ Gen Z Collection

* Dedicated streetwear section
* Gradient promotional banner
* Modern youth-focused design

### 🚚 Service Features

* Free Shipping

* Secure Payments

### 📧 Newsletter Subscription

* Email subscription form
* Promotional updates section

### 🌐 Social Media Footer

* Shop links
